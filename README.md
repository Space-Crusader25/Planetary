Description
-----------

Planetary est une petite démonstration interactive en HTML/CSS/JS qui affiche des planètes et un vaisseau spatial. Le projet sert d'exemple visuel et pédagogique pour explorer des animations, la manipulation du DOM et la structuration d'actifs web.

État: W.I.P (Work In Progress)

Fonctionnalités principales
- Interface web légère basée sur `index.html` et `galaxy.html`.
- Scripts JavaScript dans le dossier `js/` gèrent la logique et les animations.
- Ressources visuelles et styles dans `assets/` (images et CSS).

Structure du projet
- `index.html` — page d'accueil / point d'entrée.
- `galaxy.html` — vue alternative/expérimentale de la galaxie.
- `js/main.js`, `js/planets.js` — logique et animation des planètes.
- `assets/css/main.css` — styles principaux.
- `assets/img/` — images utilisées (ex. spaceship.png).

Comment lancer
1. Ouvrir `index.html` dans un navigateur moderne (double-cliquer ou via un serveur local).
2. Pour développement, exécuter un serveur local (par ex. `python -m http.server`) depuis le dossier du projet et ouvrir `http://localhost:8000`.
